== Description ==
ICF Actions is a very simple Wordpress widget.  It calls 'do_action' with the specified action name when the widget is displayed.

== Requirements ==
Wordpress and some php code that is hooked into the specified action.

== Installation ==
* Download  into your plugin directory. Or simply install via Wordpress admin interface.
* Activate.
* Add the widget to your page.
